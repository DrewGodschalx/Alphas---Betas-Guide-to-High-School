Welcome to the Features Page

Here you will find all the information you need to navigate this website with ease. You may also find some cool features that you might not have known about before.

Java
Script

With the use of JavaScript, many features of this website are possible including:

  • Local Storage
  • Window.Onload
  • Theme toggling
  • Tweet Generator
  • Font selection
  • Google Translate integration

Local Storage

Local storage is used to store

information such as the selected

light mode, font and some information

from the tweet generator.

Window.Onload

Window.Onload is a function

used to execute functions saved

in local storage or elsewhere such as

light mode, font and language.

Theme toggling

Theme toggling is limited

to light and dark mode

By pressing the button

second from the top right

you are able to toggle between

the two.

Tweet Generator

The tweet generator uses

a lot of JS, the buttons,

input boxes and generation,

all use it. You can generate

a random tweet or change

individual elements by pressing

the buttons on the top row.

The add custom content button

won't be enabled until all fields

are filled.

Font selection

The font selection dropdown

uses JS to retrieve fonts by name

it then relays the font onto every

text in the website regardless

of language, or text size.

However the font in the dropdown

remains unchanged, and provides

a preview of the font.

Google Translate integration

With the use of JS the Google

Translate element retrieves

information from Google Translate

and relays it onto the website.

JS also makes the selection and

dropdown look the same as

Google Translate.
